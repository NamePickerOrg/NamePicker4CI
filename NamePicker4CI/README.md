## 介绍

NamePicker与ClassIsland联动插件

需要配合[NamePicker](https://github.com/NamePickerOrg/NamePicker)使用

PS：NamePicker版本需要大于1.0.1dev，并且在配置中启用ClassIsland联动功能

### 特性

- 让抽选结果以通知形式显示
